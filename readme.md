吃灰的树莓派2B跑 nextcloud 卡得不行，反正就在移动硬盘上存存文件，没太大需求，就自己写一个上传文件的 web 程序。

没有数据库，没有用户验证，在局域网内用。

## 用法
```bash
git clone <这个仓库>
cd <这个仓库>
npm i # 或者用yarn
node index.js # 建议用 pm2 后台运行
# 默认的路径是运行路径，可以用 --path=<路径> 指定
# 默认的端口是 8080，可以用 --port=<端口> 指定
```
